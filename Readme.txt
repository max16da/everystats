Projet "everystat"

Réaliser un module de gestion des statistiques pour qu'il puisse être intégré dans toute application web.
Ex: nombre de visites sur un site, scores pour des jeux en ligne, nombre d'articles en vente/en stock

Utilisation de la bibliothèque CherryPy

Fonctionnalités du projet
- Ajouter une statistique à suivre
	- nom statistique
	- type de statistique (int,string,...)
	- valeur initiale pour la statistique
- Définir un modèle d'affichage (template)
	- soumettre un modèle
- Obtenir un contenu
	- produit à partir du modèle d'affichage
